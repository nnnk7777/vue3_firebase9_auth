# vue3-test

Vue3 と Firebase9 を使ったサンプル

`src` ディレクトリ内に，以下の形式の`firebase.config`を用意する

**src/firebase.config**

```
export default {
  apiKey: "****************",
  authDomain: "****************",
  projectId: "****************",
  storageBucket: "****************",
  messagingSenderId: "****************",
  appId: "****************",
}
```
